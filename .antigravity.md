# Antigravity Project Instructions

## Project Organization
- **Functions and Scripts**: All Python scripts and source code must be kept in the `scripts/` directory.
- **Data Collections**: Any source data or input files (e.g., `playlist_items.txt`) should reside in the `scripts/` or a dedicated `data/` directory.
- **Transcripts**: Individual transcript files must be saved in the `transcripts/` directory.
- **File Naming**: Use clear, descriptive names. For transcripts, use sanitized video titles with the `.md` extension.

## Transcript Formatting
- Each transcript file must include the video title as an `H1` header.
- Include the video URL at the top.
- Format transcript lines with timestamps in `[MM:SS]` format.

## General Rules
- Maintain this directory structure for all future additions.
- If generating a new function or script, ensure it is placed in the correct subfolder, not the root.
